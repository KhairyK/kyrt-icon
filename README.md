# KYRT Icon
---
**Versi**: 1.0.0

**Tipe**: Icon

**Lisensi**: MIT

**Bahasa Pemrograman**: CSS + SVG Combine

**Deskripsi**: *KYRT Icon Adalah Sebuah Icon Yang Memakai Bahasa Pemrograman CSS Dan SVG(XML).*

## Cara Memakai Nya
1. Dengan **NPM**
```bash
# jalankan ini ke Terminal Kalian
npm install kyrt-icon
```

2. Dengan Git Clone
```bash
# Jalankan ini ke terminal kalian
git clone https://github.com/KhairyK/kyrt-icon.git
```

3. Download ZIP

Silakan Pencet Tombol ***Code*** dan klik text **Download ZIP**

4. CDN Link(kyrtcdn)

Silakan Copy Dan Paste Ke Tag HTML `<head>` Kalian. 
```html
<link rel="stylesheet" href="https://cdn.kyrt.my.id/libs/1.0.1/kyrt-icon/dist/all.min.css">
```

Contoh Penggunaan(0.0.1):
```html
  <i class="mai icon-home" style="color:#4CAF50;"></i>
  <i class="mai icon-user" style="color:#2196F3;"></i>
  <i class="mai icon-settings" style="color: #cccccc;"></i>
  <i class="mai icon-folder" style="color:#FFB300;"></i>
  <i class="mai icon-html5" style="color: #FF1010;"></i>
  <i class="mai icon-js" style="color: #F0B40D"></i>
  <i class="mai icon-wa" style="color: #008000;"></i>
  <i class="mai icon-exit"></i>
  <i class="mai icon-window-os" style="color: #2172FF"></i>
  <i class="mai icon-youtube" style="color: #F72801"></i>
  <i class="mai icon-youtube-short" style="color: #F72801"></i>
  <i class="mai icon-tiktok"></i>
  <i class="mai icon-time"></i>
  <i class="mai icon-paper-pencil"></i>
  <i class="mai icon-user-2" style="color: #4692FA;"></i>
  <i class="mai icon-chair"></i>
```

Bisa Juga `mai` Diganti Dengan `i`, Misal:
```html
  <i class="i icon-chair"></i>
  <i class="i icon-paper-pencil"></i>
```
