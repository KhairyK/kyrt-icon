# Kontribusi untuk KYRT Icon

Terima kasih sudah ingin berkontribusi pada KYRT Icon!  
Proyek ini bertujuan menyediakan ikon yang bersih, konsisten, dan ringan untuk web.

## Pedoman Kontribusi

### 1. Aturan Desain Ikon

Pastikan ikon mengikuti standar berikut:

- Gaya: **minimalis, bersih, dan modern**
- Format file: **SVG**
- Ketebalan garis: **2px** (atau konsisten dengan ikon lain)
- ViewBox: `0 0 24 24`
- Ikon harus seimbang dan rata
- Hindari titik anchor yang berlebihan

> *Catatan:* Gunakan path sederhana agar performa optimal.

---

### 2. Struktur Folder & Penamaan File

- Semua ikon berada di folder `icons/`
- Gunakan format **kebab-case** untuk nama file

Contoh:

icons/panah-kiri.svg icons/kalender-centang.svg

---

### 3. Cara Mengirim Kontribusi

1. Fork repository ini
2. Buat branch baru

git checkout -b tambah-ikon-baru

3. Tambahkan ikon Anda
4. Commit dengan pesan yang jelas

feat(icon): tambah ikon kalender-centang

5. Buka **Pull Request**, sertakan:
- Preview ikon
- Penjelasan singkat
- Screenshot contoh penggunaan (opsional)

---

### 4. Pemeriksaan Sebelum Submit

Pastikan:

- ✅ SVG sudah dioptimasi (`svgo` atau alat lain)
- ✅ Gaya ikon konsisten
- ✅ Pesan commit jelas
- ✅ Tidak mengganggu ikon yang sudah ada

---

### 5. Diskusi Kontribusi

Jika ragu atau ingin berdiskusi dulu, silakan buka **Issue** terlebih dahulu.

---

### Lisensi

Dengan berkontribusi, Anda setuju bahwa kontribusi Anda dirilis dengan lisensi yang sama.
